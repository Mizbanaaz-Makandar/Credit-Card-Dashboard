1. Credit Card Dasboard
Credit Card Analytics: Transaction Insights Dashboard
A dynamic and interactive Power BI dashboard built to analyze credit card transaction data, identify spending trends, detect anomalies, and visualize customer payment behavior for improved financial insights.

2. Short Description / Purpose
The Credit Card Transaction Insights Dashboard is a visually engaging Power BI report designed to help financial analysts and decision-makers explore transaction data in depth.
It highlights key metrics such as total transactions, spending categories, customer demographics, card types, merchant performance, and fraud indicators.
This dashboard assists banks, fintechs, and analysts in identifying trends, monitoring customer activity, and ensuring transaction security.

4. Key Insights / Features
-Real-time overview of total transactions, revenue, and spending patterns.
-Category-wise spending visualization (e.g., travel, groceries, entertainment).
-Customer segmentation by age, region, and income group.
-Fraud detection indicators (suspicious transactions, outliers).
-Comparison between card types (credit, debit, premium, etc.).
-Time-based analysis of transactions (daily, monthly, quarterly).
-Dynamic filters for drill-down exploration.
